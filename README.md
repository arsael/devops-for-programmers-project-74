## Requirements
* Docker: 28.2.2+

## Commands
To setup

```bash
make setup
```

To test

```bash
make test
```

To run dev
```bash
make dev
```

To build prod

```bash
make prod-build
```

To push docker image

```bash
make prod-push
```
